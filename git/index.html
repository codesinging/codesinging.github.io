# 配置

Git 配置文件为 .gitconfig , 在用户主目录下为全局配置，项目目录下为项目配置。

# 显示当前 Git 配置

git config --list

# 编辑 Git 配置文件

git config -e [--global]

# 设置用户名和邮箱

git config [--global] user.name "<name>"
git config [--global] user.email "<email>"
